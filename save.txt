 Harika! CLI'niz başarıyla çalışıyor. İşte terminalde kullanabileceğiniz yöntemler:

  1. Doğrudan Node ile çalıştırma

  # Help göster
  node bin/cli.js --help

  # Version göster
  node bin/cli.js --version

  # Veya dist klasöründen direkt
  node dist/index.js --help

  2. Global komut olarak kullanma (Önerilen)

  Projeyi global olarak linkleyin:

  cd /Users/forwardie-m1/Desktop/demo-app
  npm link

  Sonra artık her yerden demo-app komutu ile kullanabilirsiniz:

  demo-app --help
  demo-app --version

  3. NPM'e yayınladıktan sonra

  NPM'e yayınladığınızda, diğer kullanıcılar şöyle kurabilir:

  # Global kurulum
  npm install -g demo-app

  # Kullanım
  demo-app --help

  Ya da npx ile:

  npx demo-app --help

  Şu anda test sonuçları:
  - ✓ --help komutu çalışıyor
  - ✓ --version komutu çalışıyor (v1.0.0)
  - ✓ Argüman olmadan çalıştırınca help gösteriyor

  Hangi yöntemi kullanmak istersiniz? npm link yapayım mı ki demo-app komutu ile direkt kullanabilesiniz?
